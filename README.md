# SocialPulse Dashboard

A modern social media dashboard built with React.js, Tailwind CSS, and Recharts. Manage posts, comments, analytics, notifications, profile, and settings from one place.

## Key Features

- **Mock Authentication** – Login/signup with protected routes
- **Dark/Light Mode** – Theme toggle with localStorage persistence
- **Post Management** – Create, like, save, and live search posts
- **Analytics** – Interactive charts + CSV export
- **Inline Profile Editing** – Edit name, bio, location, etc.
- **Settings Panel** – Privacy, notifications, display, language
- **Fully Responsive** – Works on mobile, tablet, desktop

## Tech Stack

React.js, Tailwind CSS, Framer Motion, Recharts, React Router, Context API, LocalStorage

## Setup
npm install
npm start
